# Cypress Task 1 – Selectors

## 📌 Description
This task focuses on practicing **best selectors in Cypress** by writing end-to-end tests for the **Sign Up page** of the Conduit demo application.

The goal is to verify the visibility and correctness of key UI elements using clean, stable, and QA-friendly selectors.

---

## 🌐 Tested Website
https://demo.productionready.io/#/register

---

## ✅ Covered Test Cases
- Check Conduit logo in the navbar
- Check Home button
- Check Sign up button in the navbar
- Verify "Sign up" title in the middle of the page
- Verify Username input field
- Verify Email input field
- Verify Password input field
- Verify Sign up submit button
- Verify Conduit logo in the footer
- Verify footer attribution text
